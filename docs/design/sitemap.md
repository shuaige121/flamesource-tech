## 全局设计规范
- 主色: #e25822 (Flame Orange)
- 辅色: #ff7b3a (Light Flame), #1a1f35 (Deep Navy)
- 背景: #080b16 (Dark Base) → #0f1320 (Section Alt)
- 文字: #f0f2f5 (Primary), #9ca3af (Secondary), #6b7280 (Muted)
- 字体: Inter (body), JetBrains Mono (code/mono)
- 间距系统: 4px base unit, sections 120px padding
- 组件库: 纯 CSS，无框架依赖
- 部署: Cloudflare Pages (纯静态 HTML)

## 设计风格
- 深色主题，精致专业（面向新加坡政府客户）
- 温暖渐变强调色，传达"火焰"品牌
- 简洁克制，无花哨动画
- 强调可信度和技术实力
